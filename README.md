## ğŸ¤ Contributing

Contributions are welcome! Please read the [Contributing Guidelines](CONTRIBUTING.md) first.

### How to contribute

1. Fork this repository
2. Create a new branch (`git checkout -b feature/add-new-tool`)
3. Add your tool to the appropriate category in the English version (`i18n/README.en.md`)
4. Submit a Pull Request

### Translation contributions

If you want to improve translations or add a new language:

1. Copy `i18n/README.en.md` as a template
2. Translate all section headers and descriptions
3. Keep tool names, URLs, and technical terms in English
4. Submit a PR with the new translation
